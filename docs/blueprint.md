# **App Name**: TradeZone

## Core Features:

- Listing Creation: Users can create listings for items they want to sell, including descriptions, photos, and pricing.
- Browse Listings: Users can browse listings by category, location, and keywords.
- Search Functionality: Implement a robust search feature to quickly find specific items.
- User Authentication: Secure user accounts with registration, login, and profile management.
- Image Generation for Placeholder: If the user does not upload an image, use generative AI to create a sample image for the item.
- Seller Rating: Rate seller performance after successful sales. Sellers can see their aggregate score, which acts as a tool to facilitate safe transactions.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Monocolo (#00A49F) to highlight key actions and success states.
- Headline font: 'Poppins', a geometric sans-serif, for a precise and contemporary look.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, neutral look.
- Use clear, modern icons to represent different product categories.
- Grid-based layout with a focus on clear, high-quality product images.
